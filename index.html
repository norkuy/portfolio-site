Stir It Up

Keep your coffee recipes in one place. Stir it up allows the user to create, edit, delete, and view their coffee recipes. Users can also create their own accounts for storing recipes. Recipes are saved in a Mongo Database on MLab. Node/Express is used on the back-end for handling requests to the custom API and jQuery is used on the front-end.

Tech Used

  • Express
  • Node
  • jQuery
  • JavaScript
  • SCSS
  • MongoDB

Alert Me

Get alerted when a product changes in price or when you're waiting for some important news to hit your favorite site. The front-end uses React for the UI and Node/Express is used on the back-end for handling the API. The server also runs periodically to check on the status of subscriptions. This has been my most ambitious project so far.

Tech Used

  • React
  • Express
  • Node
  • JavaScript
  • SCSS
  • MongoDB

Music Match

Music Match provides music recommendations based on your favorite artists. Type in an artist and Music Match will provide you with several alternatives to check out. You'll also be provided with tour information, so you'll know if the artist is playing near you!
